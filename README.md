
# Machine Learning

## `cs4641-hw1`

  - **Linear algebra:** determinants, inverses, eigenvalues/eigenvectors
  - **Probability:** expectation, covariance, correlation, independence
  - **Optimization & MLE:** KKT conditions; maximum likelihood (discrete, exponential)
  - **Information theory:** entropy, mutual information proofs; basic algorithmic complexity

## `cs4641-hw2`

  - **Clustering:** KMeans/KMeans++, clustering metrics, hierarchical clustering
  - **Density models:** EM algorithm; Gaussian Mixture Models (sampling, image compression)
  - **Semi-supervised:** messy-data cleaning with KNN and EM

## `cs4641-hw3`

  - **Matrix methods:** SVD for audio compression; SVD recommender
  - **Dimensionality reduction:** PCA eigenfaces
  - **Regression:** polynomial features, ridge regularization; GD/SGD/MBGD; cross-validation
  - **Classification:** Naive Bayes; logistic regression for text sentiment
  - **Diagnostics:** feature selection (forward/backward); class imbalance via ROC/AUC and SMOTE

## `cs4641-hw4`
  - **Neural nets:** two-layer NN with softmax/softsign, dropout, cross-entropy, backprop; GD/mini-batch/Adam
  - **Computer vision:** CNN image classification with augmentation/regularization
  - **Kernels:** SVM with custom and RBF kernels
  - **Sequences:** RNN/LSTM for next-character prediction
